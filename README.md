This project builds a binary classification model to predict passenger survival on the Titanic using the Kaggle dataset. It covers the full ML pipeline — from exploratory data analysis and missing value treatment to feature encoding, scaling, and model training. Two classifiers were compared: Logistic Regression (78.1% accuracy) and Random Forest (74.7% accuracy), with Logistic Regression emerging as the stronger baseline. Key findings include the strong predictive power of fare, sex, and passenger class.
